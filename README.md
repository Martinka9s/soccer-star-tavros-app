# Soccer Star Tavros - 5x5 Football pitch booking system

A modern web application for managing 5x5 football pitch bookings.

## Features

- User authentication
- Booking calendar for 2 pitches
- Admin approval system
- Real-time notifications
- Bilingual support (English/Greek)
- Responsive design

## Tech Stack

- React + TypeScript + Vite
- Firebase (Auth + Firestore)
- Tailwind CSS
- Deployed on Vercel

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Create `.env` file with Firebase credentials
4. Run: `npm run dev`

## Environment Variables
```
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```
